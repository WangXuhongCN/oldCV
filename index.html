Work Experiences

  • June 2020 -- Present, Research Intern in Alipay (Alibaba Group), Hangzhou, China.

    Dynamic Graph Based Fraud Detection Project

    With the rapid growth of financial services, fraud detection has been a very important problem to guarantee a healthy environment for both users and providers. Recently, lots of attempts utilize Graph Neural Network to capture community relationship and locate fraudsters. However, they usually compose graph in a static way, ignoring the dynamic characteristics of financial transactions. In this work, we use attention mechanism to aggregate static long-term relationships and dynamic short-term relationships, effectively improving the effectiveness of fraud detection.

  • Apr. 2020 -- June 2020, Research Intern in Microsoft Research Aisa, Beijing, China.

    Tree-structured Graph Based Issue Detection in Cloud

    In cloud systems, incidents affect the availability of services and require quick mitigation actions. Engineers often examine the log messages to perform fault diagnosis. However, the stacked call mechanism and the overwhelming details in log data make the manual diagnosis process time consuming and error prone. We propose Tree LSTM based performance issue detection method in cloud infrastructures. In the cloud service systems, each session is formed by diverse activities and stacked calling relationships. We treat activities as nodes and calling relationships as edges, afterwards, Tree LSTM can be used to diagnosing the incidents of a cloud system.

  • Jul. 2019 -- Dec. 2019, Deep learning Intern in Intel Asia-Pacific R&D Co., Shanghai, China.

    We developed an assisted diagnosis system for cervical cancer slices (Pap Smear), using unsupervised image anomaly detection technology to enhance the performance of supervised Resnet-50. The model can improve the diagnosis speed and reduce the rate of missed diagnosis.

  • May. 2018 -- Jan. 2019, Joint Project Engineer for China Southern Power Grid Co., Guangdong, China.

    We generalize anomaly detection technology to the smart power grid. I built a real time online fault detection system to monitor the status of the transformer substation, using RPCA algorithm, SQL database and Flask web development framework.
